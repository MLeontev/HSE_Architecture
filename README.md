## Онлайн-тренажер для подготовки разработчиков к техническим собеседованиям

Веб-приложение для помощи разработчикам в подготовке к техническим собеседованиям. Позволяет проводить пробные собеседования в условиях, приближенных к реальным, помогая кандидатам оценить свой уровень, выявить слабые места и снизить стресс перед настоящими собеседованиями

**Пользователи:** Кандидаты-разработчики (от Junior до Senior). Потенциальный охват - тысячи пользователей, учитывая высокую конкуренцию на IT-рынке труда

**Требования:**

- Кандидат выбирает параметры собеседования: грейд (Junior, Middle, Senior), язык программирования и специализацию (Backend, Frontend, Fullstack и т.д.)

- Система автоматически формирует набор заданий для каждой сессии, выбирая вопросы и задачи из базы данных

- Набор включает:

  - теоретические вопросы с открытым ответом
  - алгоритмические задачи с написанием кода в браузере

- На выполнение всех заданий выделяется ограниченное время

- После ответа на теоретический вопрос кандидат получает эталонный ответ и и обратную связь от ИИ-модели

- Написанный код проверяется автоматически: кандидат получает вердикт и информацию о прохождении тест-кейсов

- По окончании сессии кандидат получает итоговый фидбек: общую статистику (затраченное время, количество решенных задач) и детализацию по каждому заданию

**Дополнительный контекст:**

- Ключевое ценностное предложение — создание условий, моделирующих реальное собеседование, включая лимит времени на решение задач, получение обратной связи, что обеспечивает более точную оценку готовности кандидата

- Планируется интеграция со сторонними AI-API (например, GigaChat) для анализа текстовых ответов и, возможно, программного кода

- Важна масштабируемость системы для одновременного проведения множества сессий, особенно в "часы-пик" (вечера, выходные)

- В будущем возможна монетизация через подписку (premium-доступ к расширенной базе задач и детальному анализу) и white-label решение для IT-компаний
